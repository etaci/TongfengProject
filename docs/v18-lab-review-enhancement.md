# V18 化验单复盘增强

## 本轮目标

V18 把“化验单识别完成”推进到“能看懂变化、知道下一步怎么跟进”。

本轮新增能力：

- 基于某一份化验单生成复盘结论
- 自动对比上一份报告中的同类指标
- 重点判断本次尿酸是否达到个人目标值
- 输出复查建议、关键变化和下一步三件事
- 在前端化验单区块直接展示复盘结果

## 新增接口

### 查看化验单复盘

- `GET /api/v1/lab-reports/{reportId}/review`

返回内容包括：

- 当前报告风险等级
- 与上一份报告的对比对象和间隔天数
- 本次尿酸值、个人目标值、是否达标
- 关键指标对比列表
- 关键变化摘要
- 复查建议
- 下一步三件事
- 可信边界提示

## 复盘逻辑

当前版本重点做两层判断：

- 尿酸是否高于个人目标值
- 与上一份报告相比，关键指标是上升、下降还是基本持平

其中尿酸会优先识别：

- `UA`
- `uric`
- `尿酸`

如果当前没有上一份报告，会明确提示“缺少基线”。

如果当前没识别到尿酸指标，也会明确提示需要人工核对，而不是静默给出目标结论。

## 前端变化

- 助手页的“最近一次化验单结果”卡片现在会补充展示：
  - 目标尿酸
  - 本次尿酸
  - 与上次间隔
  - 复盘摘要
  - 关键变化
  - 指标对比
  - 复查建议
  - 下一步三件事
  - 可信边界

此外，刷新后也会自动回显最近一次化验单结果和对应复盘。

## 安全边界

- 复盘结论只基于已识别指标和当前档案目标值生成
- 不能替代医生诊断、处方或正式检验报告解释
- OCR 漏识别关键指标时，系统会提醒重新上传或人工核对
